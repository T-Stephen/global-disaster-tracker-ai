### 🚀 Executive Summary

Global crisis response is historically hampered by delayed information and reactive planning. The **Global Disaster Tracker AI** is an enterprise-grade intelligence platform engineered to solve this. By aggregating live geospatial data and processing it through predictive AI models, this system provides emergency coordinators with a real-time interactive dashboard to track current anomalies and forecast future disaster trajectories.

### ⚙️ System Architecture

* **Predictive AI Engine (Backend):** Built on **Python**, this module evaluates historical environmental datasets and live anomalies to generate probabilistic models for impending natural disasters.
* **Geospatial Command Center (Frontend):** A high-performance **ReactJS** interface featuring interactive mapping (via Mapbox/Leaflet), allowing users to visualize complex crisis data in a fluid, low-latency environment.
* **Live Telemetry & Analytics:** An integrated dashboard that transforms raw global APIs into actionable, high-contrast analytics for rapid decision-making.

### 🛠️ Local Deployment (Quick Start)

To initialize the global tracking matrix locally, you will need to spin up both the backend engine and the frontend command center:

```bash
# 1. Clone the repository
git clone [https://github.com/T-Stephen/global-disaster-tracker-ai.git](https://github.com/T-Stephen/global-disaster-tracker-ai.git)
cd global-disaster-tracker-ai

# 2. Initialize the Python AI Backend
cd backend
pip install -r requirements.txt
python app.py

# 3. Initialize the React Frontend (in a new terminal)
cd ../frontend
npm install
npm start
